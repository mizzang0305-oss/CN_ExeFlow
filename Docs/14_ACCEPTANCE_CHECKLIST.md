# 설계 잠금 완료 체크리스트

## A. 제품 정의
- [ ] 본 시스템의 목적이 ‘대표 지시 이행관리’로 고정되었는가
- [ ] 회의록 앱이 아닌 실행 통제 시스템으로 정의되었는가

## B. 범위
- [ ] MVP 필수 범위가 정리되었는가
- [ ] 제외 범위가 정리되었는가

## C. 역할/권한
- [ ] 역할이 CEO / SUPER_ADMIN / DEPARTMENT_HEAD / STAFF / VIEWER로 고정되었는가
- [ ] 역할별 권한이 정의되었는가

## D. 상태
- [ ] 상태가 NEW / IN_PROGRESS / COMPLETION_REQUESTED / DELAYED / COMPLETED / REJECTED로 고정되었는가
- [ ] 상태 전환 조건이 정의되었는가
- [ ] 종결 승인 기준이 정의되었는가

## E. DB
- [ ] 핵심 테이블이 확정되었는가
- [ ] 관리번호 고유성 규칙이 반영되었는가
- [ ] 감사 로그 보존 원칙이 반영되었는가

## F. 화면
- [ ] 대표 대시보드 정의 완료
- [ ] 지시사항 목록/상세 정의 완료
- [ ] 부서 실행보드 정의 완료
- [ ] 행동 로그 등록 화면 정의 완료
- [ ] 주간 결산 화면 정의 완료

## G. 리포트
- [ ] KPI 계산식이 정의되었는가
- [ ] 대표 1면 요약 원칙이 정의되었는가

## H. 프롬프트
- [ ] Stitch UI용 프롬프트 준비 완료
- [ ] Gemini 기본기 구축용 프롬프트 준비 완료
