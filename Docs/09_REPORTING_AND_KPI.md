# 리포트 및 KPI 정의

## 1. 전사 KPI
- 총 관리 건수
- 금주 신규 건수
- 진행중 건수
- 지연 건수
- 종결 건수
- 종결률
- 기한 준수율

## 2. 계산식
- 총 관리 건수 = 전주 이월 건수 + 금주 신규 건수
- 종결률 = 종결 건수 / 총 관리 건수 * 100
- 기한 준수율 = 기한 내 종결 건수 / 전체 종결 건수 * 100

## 3. 부서 KPI
- 부서별 관리 건수
- 부서별 지연 건수
- 부서별 종결 건수
- 부서별 종결률
- 부서별 평균 종결 소요일
- 증빙 등록률
- 업데이트 준수율

## 4. 주간 결산 포함 항목
- 금주 신규 등록
- 현재 진행중
- 일정 지연
- 금주 종결
- 부서별 요약
- 대표 확인 필요 사항
- 주요 리스크

## 5. 대표 1면 요약 원칙
- 수치 우선
- 지연/리스크 우선
- 중요도 높은 건 우선
- 상세는 링크 또는 drill-down

## 6. 향후 확장 KPI
- 재지시율
- 반려율
- 무응답 건수
- 지시 유형별 평균 완료기간
- 대표 재확인 빈도
