# 역할 및 권한 정의

## 역할 목록
1. CEO
2. SUPER_ADMIN (TF 총괄)
3. DEPARTMENT_HEAD
4. STAFF
5. VIEWER

## CEO
### 가능
- 전사 대시보드 조회
- 모든 지시 상세 조회
- 핵심 건 승인/코멘트
- 종결 현황 확인
### 제한
- 일반 실무 로그 직접 편집은 기본 비권장

## SUPER_ADMIN
### 가능
- 지시 등록/수정
- 관리번호 발급
- 부서 배정
- 상태 강제 조정(권한 기반)
- 완료 요청 승인/반려
- 주간 결산 생성
- 기준정보 관리
### 제한
- 없음(운영 총괄)

## DEPARTMENT_HEAD
### 가능
- 자기 부서 건 조회
- 담당자 배정
- 부서 상태 업데이트
- 로그/증빙 검토
- 완료 요청 제출
### 제한
- 타 부서 건 수정 불가
- 기준정보 관리 불가

## STAFF
### 가능
- 본인 또는 배정된 과업 로그 등록
- 사진/문서 증빙 업로드
- 진행 상황 업데이트
### 제한
- 종결 승인 불가
- 관리번호 수정 불가
- 타 부서/타 담당자 핵심 정보 수정 불가

## VIEWER
### 가능
- 조회만 가능
### 제한
- 등록/수정/삭제 불가
