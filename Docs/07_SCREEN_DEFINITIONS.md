# 화면 정의서

## 1. 대표 메인 대시보드
### 목적
대표가 전사 현황을 1분 안에 파악

### 상단 KPI
- 총 관리 건수
- 금주 신규
- 진행중
- 지연
- 금주 종결
- 종결률
- 기한 준수율

### 중단 섹션
- 부서별 현황 차트
- 지연 상위 부서
- 마감 임박 건
- 금주 종결 추이

### 하단 섹션
- 대표 확인 필요 건
- 최근 활동 피드
- 중요도 높은 지연 건

## 2. 지시사항 목록
### 컬럼
- 관리번호
- 제목
- 소관부서
- 책임자
- 상태
- 중요도
- 지시일
- 종결예정일
- 지연여부

### 기능
- 검색
- 상태 필터
- 부서 필터
- 중요도 필터
- 신규 등록

## 3. 지시사항 상세
### 섹션
- 기본 정보
- 지시 원문
- 부서 배정 현황
- 담당자/세부 과업
- 상태 이력
- 행동 로그 타임라인
- 증빙 파일
- 완료 요청 / 승인 / 반려 영역

## 4. 부서 실행보드
### 목적
부서장이 자기 부서 건을 빠르게 처리

### 영역
- 우리 부서 KPI
- 담당자별 배정 건수
- 상태별 보드
- 완료 요청 전 점검
- 증빙 미등록 필터

## 5. 행동 로그 등록 화면
### 입력 항목
- 관련 지시사항
- 관련 과업
- 로그 타입
- 행동 요약
- 상세 내용
- 조치 일시
- 다음 액션
- 리스크 메모
- 사진 첨부
- 문서 첨부

## 6. 주간 결산 리포트 화면
### 포함 항목
- 주간 범위
- 총 관리 건수
- 신규
- 진행중
- 지연
- 종결
- 종결률
- 기한 준수율
- 부서별 요약
- 금주 지연 상세
- 금주 종결 상세
